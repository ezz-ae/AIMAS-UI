Forbidden Patterns
------------------
- Ranking, recommendations, or personalization.
- Copying protocol content into AIMAS-UI source files.
- Storing or replaying raw intent payloads.
- Combining protocol logic with UI logic.
- Introducing AI/LLM inference inside AIMAS core.
- Persuasive copy that explains the protocol instead of proving it by structure.
- Any feature that removes or weakens the Free Baseline Path.
